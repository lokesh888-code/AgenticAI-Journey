# Student File Manager

A beginner Python mini project using file handling, `os`, `pathlib`, and `json`.

## Features
- Write student records to JSON
- Read JSON records
- Display current folder contents
- Generate a simple file report

## How to run
1. Run `python json_write.py`
2. Run `python json_read.py`
3. Run `python pathlib_demo.py`
4. Run `python os_operations.py`
5. Run `python file_report.py`

## Learning outcome
This project helps practice file handling, structured data storage, and folder inspection in Python.
